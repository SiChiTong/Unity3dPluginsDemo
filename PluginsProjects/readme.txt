UnityPluginsCLR:
Visual Studio 2015 基于 .net framework 3.5 的托管plugin .dll动态库，Windows平台

UnityPluginsNative：
Visual Studio 2015 纯C的Native plugin .dll动态库，Windows平台

UnityPluginsAndroid：
Android Java，.jar动态库

UnityPluginsIOS：
IOS Object-c，.a静态库

UnityPluginsMacOSX：
Mac OS X Object-c，.bundle包



【Plugins类别】：动态库、静态库
【Windows】：.dll(托管、非托管)
【Linux】：.so
【Mac OS X】：.bundle
【Android】：Java、C/C++；.jar、.so
【IOS】：只能是静态库.a、object-c
